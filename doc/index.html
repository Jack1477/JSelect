下面这两个是被美化的组件

下面这两个是没有被美化的组件

以上两个美化后组件的支持性能

第一个select组件的代码

<select class="mySelect">
    <option disabled value="1">Jack</option>
    <option selected value="2">Jun</option>
    <option value="3">C</option>
</select>

让我们尝试修改select的value

document.querySelector('.mySelect')._value=3

当你点击了上面这个按钮,会发现第一个UI-Select框的选项被改变了
同理获取value document.querySelector('.mySelect').value 同样可行

让我们尝试绑定select的onchange Event 

document.querySelector('.mySelect').onchange = function (value, displayHTML) {
    console.log("Change!");
    console.log(value, displayHTML)
}


当你点击了上面这个按钮,再去改变第一个select UI ,观察控制台输出
onchange 接受两个参数 (value,displayHTML)
